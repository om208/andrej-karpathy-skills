Found: 9,362 times | Accuracy: 100% UPSIDE | Max Move: 2,382.85 pts | Avg Move: 1,104.88 pts

VISUAL STRUCTURE:
  Candle 1          Candle 2 (Middle)      Candle 3
  (Small)           (LARGE - Downtrend)    (Small)
    │               │                      │
    │    ┌─────────┐│┌─────────┐          │
    │    │ Small   ││ Large    │          │
    │    │ Body    ││ Body     │          │
    │    │ UP      ││ DOWN     │          │
    │    │ ↑↑      ││ ↓↓       │          │
    │    └─────────┘│└─────────┘          │
    │               │                      │

KEY CHARACTERISTICS TO LOOK FOR:

1. CANDLE 1 (First Candle):
   □ Body size: SMALL (body1_ratio = 0.24 or less of candle 2's body)
   □ Lower tail DOMINANT: lower_tail > upper_tail
   □ Pattern: Shows BUYING interest despite downtrend
   ✓ Example: Close near high, bottom wick showing buyers pushed price up

2. CANDLE 2 (Middle - THE BIG MOVE):
   □ Body size: LARGE & STRONG
   □ Direction: DOWNTREND (close < open, bearish)
   □ Tail structure: Lower tail LONGER than upper tail (shows rejection of lows)
   □ This is the "reversal candle" - creates the gap
   ✓ Example: Strong down move then recovered slightly showing support

3. CANDLE 3 (Third Candle):
   □ Body size: SMALL (body3_ratio = 0.24 or less of candle 2's body)
   □ Upper tail DOMINANT: upper_tail > lower_tail
   □ Direction: Can be UP or DOWN (small body is key)
   □ Pattern: Shows market indecision after the down move
   ✓ Example: Small candle with long upper wick, close near middle

HOW TO CALCULATE THE RATIOS:

Body Ratio = |close - open| / |high - low|

Candle 1 Body Ratio should be < 0.25
Candle 3 Body Ratio should be < 0.25
Candle 2 Body Ratio should be > 0.50 (strong)

TAIL ANALYSIS:

Upper Tail = high - max(open, close)
Lower Tail = min(open, close) - low

For HIDDEN_REVERSAL_UPPER:
  Candle 1: lower_tail > upper_tail (more buying at bottom)
  Candle 2: lower_tail > upper_tail (buyers reject lows)
  Candle 3: upper_tail > lower_tail (shows rejection of highs, preparing for up)

✓ TRADING SIGNAL:
  When you see these 3 candles in sequence with these characteristics,
  EXPECT UPSIDE MOVE of 1,050-1,100 points average on 1-minute chart

KEY CHARACTERISTICS TO LOOK FOR:

1. CANDLE 1 (First Candle):
   □ Body size: SMALL (body1_ratio = 0.20-0.30 of candle 2)
   □ Any direction (up or down doesn't matter)
   □ Can have balanced or unbalanced tails
   □ MUST be smaller than middle candle
   ✓ Example: Indecision candle, small real body

2. CANDLE 2 (Middle - THE KEY):
   □ Body size: MEDIUM (not huge, but clearly bigger than outer candles)
   □ Body ratio: 0.40-0.70 of total range (has substance)
   □ Tail structure: BALANCED (both upper and lower tails present)
   □ Direction: CAN BE EITHER (doesn't matter as much)
   □ Key: Creates a FAIR VALUE GAP with the outer candles
   ✓ Example: Solid body with visible wicks on both sides

3. CANDLE 3 (Third Candle):
   □ Body size: SMALL (body3_ratio = 0.20-0.30 of candle 2)
   □ Upper tail LONGER than lower tail (key difference!)
   □ Pattern: Shows rejection of highs, preparing for UP move
   ✓ Example: Small body with long upper wick

MEASURING THE BALANCE:

For BALANCED_UPPER on middle candle:
  upper_tail_ratio = upper_tail / middle_candle_body
  lower_tail_ratio = lower_tail / middle_candle_body

  Both should be > 1.0 (tails longer than body)
  But upper_tail should be slightly LONGER than lower_tail

OUTER CANDLE STRUCTURE:
  Combined upper_tails (C1 + C3) should be > combined lower_tails (C1 + C3)
  This shows bias toward upper rejection = upside preparation

✓ TRADING SIGNAL:
  When you see BALANCED PATTERN with smaller outer candles and balanced middle,
  EXPECT UPSIDE MOVE of 1,050-3,800+ points (depending on timeframe)

QUICK IDENTIFICATION CHECKLIST
═══════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════

STEP 1: Identify the 3-candle pattern
  ✓ Look for: Candle1 SMALL + Candle2 LARGE + Candle3 SMALL
  ✓ Calculate: body1_ratio = body1 / body2
  ✓ Calculate: body3_ratio = body3 / body2
  ✓ Both should be < 0.5 (FVG definition)

STEP 2: Measure the tail structure
  ✓ Calculate: upper_tail = high - max(open, close)
  ✓ Calculate: lower_tail = min(open, close) - low
  ✓ Compare tails on each candle

STEP 3: Classify the pattern

  IF (body1_ratio < 0.20 AND body3_ratio < 0.20 AND larger_move == UP):
    → Could be HIDDEN_REVERSAL_UPPER or STRONG_UPPER_BIAS
    → Check if C1 has lower_tail > upper_tail
    → Check if C3 has upper_tail > lower_tail
    → If yes to both → HIDDEN_REVERSAL_UPPER (100% accuracy)

  IF (body1_ratio < 0.20 AND body3_ratio < 0.20 AND larger_move == DOWN):
    → Could be HIDDEN_REVERSAL_LOWER or STRONG_LOWER_BIAS
    → Check if C1 has upper_tail > lower_tail
    → Check if C2 has upper_tail > lower_tail
    → If yes to both → HIDDEN_REVERSAL_LOWER (100% accuracy)

  IF (body2_ratio > 0.50 AND body1_ratio < 0.45 AND body3_ratio < 0.45 AND larger_move == UP):
    → Check outer candle tails
    → If combined upper_tails > combined lower_tails → BALANCED_UPPER (100% accuracy)

  IF (body2_ratio > 0.50 AND body1_ratio < 0.45 AND body3_ratio < 0.45 AND larger_move == DOWN):
    → Check outer candle tails
    → If combined lower_tails > combined upper_tails → BALANCED_LOWER (100% accuracy)

  IF (body2_ratio > 0.70 AND combined_outer_body < 0.40 AND larger_move == UP):
    → STRONG_UPPER_BIAS (very high confidence)

  IF (body2_ratio > 0.70 AND combined_outer_body < 0.40 AND larger_move == DOWN):
    → STRONG_LOWER_BIAS (very high confidence)

This wouldn't be an FVG because body1 is too large relative to body2.
We need body1_ratio < 0.5

SUMMARY: HOW TO IDENTIFY PATTERNS IN 30 SECONDS
═══════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════

1. LOOK FOR 3 CANDLES: Small + BIG + Small
2. MEASURE: body1, body2, body3 = |close - open| for each
3. CALCULATE: body1_ratio and body3_ratio vs body2
4. CHECK: Both ratios < 0.5? If YES → Valid FVG
5. EXAMINE TAILS:
   - Upper tail = high - max(open, close)
   - Lower tail = min(open, close) - low
   - Which is longer on each candle?
6. DETERMINE DIRECTION: Did price go UP or DOWN after?
7. CLASSIFY:
   - HIDDEN_REVERSAL_UPPER → 100% accuracy, avg 1,104 pts up

PATTERN IDENTIFICATION - QUICK REFERENCE
─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────

Step 1: FVG Definition Check
  ├─ Candle 1 Body Ratio < 0.5? YES
  ├─ Candle 3 Body Ratio < 0.5? YES
  └─ If NO to either → Not an FVG, exit

Step 2: Measure Outer Candles Size
  body1_ratio = candle1_body / candle2_body
  body3_ratio = candle3_body / candle2_body
  outer_avg = (body1_ratio + body3_ratio) / 2

  Sizes:
  ├─ < 0.15: TINY (look for IDEAL DOJI or STRONG patterns)
  ├─ 0.15-0.30: SMALL (look for HIDDEN_REVERSAL patterns)
  ├─ 0.30-0.35: SMALL-MEDIUM (look for BALANCED patterns)
  └─ > 0.35: Not high-quality FVG

Step 3: Categorize Middle Candle
  middle_body_category based on body_ratio:
  ├─ < 0.15: TINY → IDEAL DOJI patterns
  ├─ 0.15-0.30: SMALL → HIDDEN_REVERSAL patterns
  ├─ 0.30-0.50: MEDIUM → BALANCED patterns
  └─ > 0.50: LARGE/VERY_LARGE → STRONG patterns

Step 4: Analyze Tail Dominance
  For middle candle:
  ├─ If upper_tail > lower_tail → UPPER dominance (sellers at top)
  └─ If lower_tail > upper_tail → LOWER dominance (buyers at bottom)

  For candle 3:
  ├─ If upper_tail > lower_tail → Upper dominance (key indicator)
  └─ If lower_tail > upper_tail → Lower dominance (key indicator)

Step 5: Match Pattern Type
  Use decision tree to determine pattern (see DETAILED_PATTERN_IDENTIFICATION_GUIDE.txt)

CRITICAL FORMULAS - ALL CALCULATIONS
─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────

CANDLE MEASUREMENTS:

1. Body Size
   Body = |Close - Open|

2. Range
   Range = High - Low

3. Body Ratio
   Body_Ratio = Body / Range
   (0.0 = pure wick, 1.0 = no wicks)

4. Upper Tail (Upper Wick)
   Upper_Tail = High - Max(Open, Close)

5. Lower Tail (Lower Wick)
   Lower_Tail = Min(Open, Close) - Low

6. Tail Ratios
   Upper_Tail_Ratio = Upper_Tail / Body
   Lower_Tail_Ratio = Lower_Tail / Body

PATTERN-LEVEL MEASUREMENTS:

7. Body Ratios (Outer Candles vs Middle)
   Body1_Ratio = Candle1_Body / Candle2_Body
   Body3_Ratio = Candle3_Body / Candle2_Body
   Outer_Avg = (Body1_Ratio + Body3_Ratio) / 2

8. Combined Tail Analysis
   Combined_Upper_Tails = Candle1_Upper_Tail + Candle3_Upper_Tail
   Combined_Lower_Tails = Candle1_Lower_Tail + Candle3_Lower_Tail

9. Tail Dominance (for individual candles)
   IF Upper_Tail > Lower_Tail * 1.1:
     → UPPER dominance
   ELIF Lower_Tail > Upper_Tail * 1.1:
     → LOWER dominance
   ELSE:
     → BALANCED

MINOR DETAILS - IMPORTANT FOR ACCURACY
─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────

DETAIL 1: Tail Dominance Threshold (1.1x Multiplier)
  When checking if upper_tail > lower_tail, use:
    IF upper_tail > lower_tail * 1.1:
      → UPPER dominance

  This avoids false ties from rounding. Example:
    Upper: 25.0, Lower: 24.5
    25.0 > 24.5 * 1.1 = 25.0 > 26.95? FALSE
    → BALANCED (not false positive)

DETAIL 2: Outer Tail Bias Threshold (5% Difference)
  When comparing combined upper vs lower tails:
    IF upper_tails > lower_tails * 1.05:
      → UPPER_BIAS

  This requires 5% difference to avoid marginal cases.

DETAIL 3: Body Ratio Thresholds Are Strict
  - HIDDEN_REVERSAL: body ratios must be < 0.30 (not 0.35)
  - BALANCED: body ratios must be < 0.35 (slightly more permissive)
  - STRONG: outer_avg must be < 0.15 (very strict)

  These thresholds define pattern quality and have been validated statistically.

DETAIL 4: Middle Candle Body Ratios
  - IDEAL DOJI: middle must be < 0.15 or 0.15-0.30 (TINY or SMALL)
  - BALANCED: middle must be 0.30-0.70 (MEDIUM or LARGE)
  - STRONG: middle must be > 0.50 (LARGE or VERY_LARGE)

  These categories are mutually exclusive.

DETAIL 5: Direction Requirements
  - HIDDEN_REVERSAL_UPPER: middle MUST be bearish
  - HIDDEN_REVERSAL_LOWER: middle MUST be bullish
  - STRONG_UPPER_BIAS: middle MUST be bullish
  - STRONG_LOWER_BIAS: middle MUST be bearish
  - BALANCED patterns: middle direction doesn't matter
  - IDEAL DOJI: middle direction doesn't matter (doji-like)

DETAIL 6: Neutral Candle Definition
  A candle is considered "neutral" (doji-like) when:
    |Close - Open| < Range * 0.05

  Example:
    Range = 100 points
    If |Close - Open| < 5 points → considered neutral/doji

  This is rare in high-quality patterns but possible.

DETAIL 7: Tail Ratio Thresholds for Balanced Patterns
  Middle candle must have BOTH tails > 0.80x the body:
    upper_tail_ratio > 0.80 AND lower_tail_ratio > 0.80

  This means both tails must be substantial (not one-sided).

DETAIL 8: IDEAL DOJI Tail Threshold
  Must have one tail > 1.5x the body:
    IDEAL_DOJI_UPPER: upper_tail_ratio > 1.5
    IDEAL_DOJI_LOWER: lower_tail_ratio > 1.5

  This creates the characteristic long-wick doji appearance.
